## Fear of the Blinking Cursor

My first encounter with the terminal was a disaster. I typed the wrong `rm` command and wiped a folder. I swore it off for two years.

## The Turning Point

A friend showed me `fzf` for fuzzy file search. I was hooked instantly. From there I discovered `ripgrep`, `bat`, `exa`, and eventually `tmux`.

## Building My Config

I spent a weekend building a dotfiles repo. Zsh with `starship` prompt, aliases for common git operations, and a custom `PATH` setup. The investment paid back immediately.

## The Mindset Shift

The terminal rewards curiosity. Every tool has a man page. Every command composes with others via pipes. Once that clicks, you stop seeing it as intimidating and start seeing it as expressive.

## Where to Start

Learn `cd`, `ls`, `cat`, `grep`, `find` and basic pipes. Then customize your prompt. The rest follows naturally.
