# My Blog

A minimal, fast, SEO-optimised personal blog built with plain HTML, CSS, and vanilla JavaScript. No framework. No build step. Deployed via GitHub Pages.

---

## Features

- **Tag filtering** — click any tag to filter posts across all sections
- **Sections** — Recent, Popular, and My Favorites
- **Blog cards** — thumbnail, title, date, description, hover + shadow effects
- **Medium-style reader** — full post page with reading progress bar
- **Slug URLs** — `post.html?slug=my-post-title`
- **SEO ready** — Open Graph, Twitter Cards, JSON-LD, canonical URLs, sitemap, robots.txt
- **Blog automation** — CLI script to add new posts instantly
- **Related posts** — shows posts with matching tags at the bottom of each post
- **GitHub Pages ready** — no build step needed

## Setup: GitHub Pages

### 1. Create your repository

```bash
git init
git remote add origin https://github.com/yourusername/yourusername.github.io.git
```

> If your repo name is `yourusername.github.io`, your site will live at `https://yourusername.github.io/`.
> Otherwise it will live at `https://yourusername.github.io/repo-name/`.

### 2. Push your files

```bash
git add .
git commit -m "Initial blog commit"
git push -u origin main
```

### 3. Enable GitHub Pages

1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select `Deploy from a branch`
3. Select branch: `main`, folder: `/ (root)`
4. Click **Save**

Your site will be live in ~1 minute at `https://yourusername.github.io/`.

## Customise

### Change your name / site URL

Find and replace `yourusername` in:
- `index.html` (canonical, OG URLs)
- `robots.txt`
- `scripts/generate-sitemap.js`

Also update `Your Name` in `post.html` and `post.js`.

### Change colours / fonts

Edit the CSS variables at the top of `assets/css/style.css`:

```css
:root {
  --accent: #7c6af7;   /* Purple accent — change to any colour */
  --bg: #0e0e10;       /* Background */
  ...
}
```

## Writing Posts

### Option A: CLI automation (recommended)

Requires Node.js.

```bash
node scripts/new-post.js
```

Follow the prompts. The script will:
- Ask for title, description, tags, thumbnail, popular/favorite flags
- Ask you to paste your post HTML content
- Auto-generate a slug from the title
- Estimate read time
- Add the post to `posts/posts.json`

### Option B: Edit posts.json manually

Open `posts/posts.json` and add a new object:

```json
{
  "id": 7,
  "slug": "my-new-post-title",
  "title": "My New Post Title",
  "date": "2025-06-01",
  "description": "A short description shown on the card.",
  "thumbnail": "https://images.unsplash.com/photo-xxx?w=600&q=80",
  "tags": ["linux", "cli"],
  "popular": false,
  "favorite": true,
  "readTime": "5 min read",
  "content": "<h2>Intro</h2><p>Your HTML content here.</p>"
}
```

Fields:
| Field | Type | Description |
|-------|------|-------------|
| `id` | number | Unique ID (increment) |
| `slug` | string | URL-safe identifier (`my-post-slug`) |
| `title` | string | Post title |
| `date` | string | ISO date (`YYYY-MM-DD`) |
| `description` | string | Card subtitle |
| `thumbnail` | string | Image URL |
| `tags` | string[] | Array of tag strings |
| `popular` | boolean | Shows in Popular section |
| `favorite` | boolean | Shows in Favorites section |
| `readTime` | string | Display string (`7 min read`) |
| `content` | string | Full HTML post content |

## SEO

### Sitemap

Generate/update your sitemap before every deploy:

```bash
node scripts/generate-sitemap.js
```

### Google Search Console

1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add property → URL prefix → `https://yourusername.github.io/`
3. Verify ownership (HTML tag or DNS)
4. Submit your sitemap: `https://yourusername.github.io/sitemap.xml`

Google will start indexing your posts within days to weeks.
